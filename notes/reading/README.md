# Reading Notes

建议按主题或日期创建自己的阅读笔记文件，例如：

- `2026-07-09-introduction.md`
- `2026-07-10-main-technique.md`
- `2026-07-16-main-technique.md`
- `2026-07-17-state-machine-proof-order.md`
- `proof-reading-log.md`

每次阅读都记录日期、阅读范围、理解、疑问和下一步。

## Template

```md
# Reading Log - Topic

## YYYY-MM-DD

- Paper section/pages:
- Time spent:
- What I understood:
- Key definitions/theorems/algorithms:
- Confusing points:
- Questions for group discussion:
- Possible text for review:
- References to check next:
```
