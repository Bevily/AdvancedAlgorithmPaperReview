# Reading Notes

建议每位成员创建自己的阅读笔记文件，例如：

- `member-a-reading-log.md`
- `member-b-reading-log.md`

每次阅读都记录日期、阅读范围、理解、疑问和下一步。

## Template

```md
# Reading Log - Name

## YYYY-MM-DD

- Paper section/pages:
- Time spent:
- What I understood:
- Key definitions/theorems/algorithms:
- Confusing points:
- Questions for group discussion:
- Possible text for review:
- References to check next:
```
