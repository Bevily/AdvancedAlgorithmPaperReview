# 中文课程论文排版设计

## 目标

将当前 FLP 论文综述调整为简洁、正式的中文课程论文版式，统一所有章节标题与结构性名称的语言和风格。改动只涉及排版、标题命名和文档元信息，不改写各章节的论述内容。

## 整体风格

- 使用 A4 单栏中文论文版式。
- 全文以黑色为主，不使用装饰横线、色块、渐变或标题边框。
- 通过字号、字重、段前段后间距和自然留白区分标题层级。
- 正文字号与行距以连续阅读和纸质批注为主要考虑，保持页面信息密度适中。

## 首页与前置部分

- 主标题使用中文，内容为“一个故障进程下分布式共识不可能性”。
- 副标题说明本文为 FLP 论文综述。
- 保留作者和学号填写位置，日期使用中文格式。
- 摘要标题改为“摘要”，新增“关键词”行。
- 摘要之后另起一页放置目录，目录标题使用“目录”。

## 标题体系

- 一级章节显示为“一、引言”“二、问题与动机”等中文编号形式。
- 二级章节显示为“（一）预备定义”“（二）引理一：互不相关调度的交换性”等中文编号形式。
- 所有标题均为纯黑中文标题，不设置下划线或其他装饰元素。
- 将现有英文标题“Introduction”改为“引言”。
- 保留 FLP、Paxos、PBFT、Raft 等论文或协议名称，以及 Fisher、Lynch、Paterson 等作者姓名的英文写法。

## 中文结构名称

- 定理环境名称统一为“定理”。
- 引理环境名称统一为“引理”。
- 定义环境名称统一为“定义”。
- 参考文献标题统一为“参考文献”。
- 摘要、关键词、目录和日期均采用中文格式。

## 页面元素

- 正文页使用简洁页眉，内容为课程名称或论文综述简称。
- 首页不显示页眉。
- 页脚居中显示阿拉伯数字页码。
- 页眉页脚只承担定位功能，不使用横线或彩色装饰。

## 实施边界

- 修改 `paper/main.tex` 中的文档类、宏包配置、标题信息、摘要和全局版式定义。
- 修改 `paper/sections/introduction.tex` 的英文章节标题。
- 仅在确有需要时调整其他章节的标题命令，不改写正文段落。
- 不删除或替换现有证明图片、引用和参考文献数据。
- 不修改用户当前位于 `notes/` 下的其他工作内容。

## 验证标准

- 使用 `latexmk -xelatex -interaction=nonstopmode -halt-on-error main.tex` 成功生成 PDF。
- 日志中不存在未定义引用、未解析文献或关键 LaTeX 警告。
- 渲染 PDF 各页进行目视检查，确认标题层级一致、中文字符正常、图文无重叠、页眉页脚无冲突。
- 检查目录、摘要、参考文献和全部章节标题均采用约定的中文风格。
