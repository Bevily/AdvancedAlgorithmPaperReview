# AI Usage Records

每次使用 AI 帮助理解、查资料、改写思路或排查 LaTeX 问题，都建议记录。

不要只写“用了 AI”。要写清楚问了什么、AI 回答了什么、你如何验证、最终有没有采纳。

## File Naming

```text
YYYY-MM-DD-topic.md
```

## Template

```md
# AI Usage - YYYY-MM-DD - Topic

- Tool:
- User/member:
- Purpose:

## Prompt Summary

写出你问 AI 的问题摘要。必要时贴关键 prompt。

## AI Response Summary

总结 AI 给出的回答，不要无脑复制大段内容。

## Verification

- Sources checked:
- Paper sections checked:
- Which parts were correct:
- Which parts were wrong or unsupported:

## How It Was Used

- Adopted:
- Modified:
- Rejected:

## Related Files / Commits

- 
```
