# Work Session Notes

每次集中阅读、推导、查资料、整理大纲或修改正文时，建议新建一个文件：

```text
YYYY-MM-DD-topic.md
```

这些记录用于展示你从资料收集、逐渐理解、观点形成、草稿撰写到反复修改的过程。
